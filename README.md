# public-assets-jsdelivr


- To use the js-delivered CDN network, you can use the markdown entry, something like this.
```
![whiteboard notes](https://cdn.jsdelivr.net/gh/sathwickreddyy/public-assets-jsdelivr@main/zz_image_dump/IMG_4526.jpg)
```
- Wrap the folder and the path location at the end. 
```
https://cdn.jsdelivr.net/gh/sathwickreddyy/public-assets-jsdelivr@main/<path>
```
